simple-char-count
=================

JQyuery Plugin - Simple Character Counter

requires JQuery

usage:

&lt;form&gt;
&lt;textarea id="message"&gt; &lt;/textarea&gt;
&lt;/form&gt;

&lt;script type="text/javascript"&gt;
$(function(){
    $("#message").charCounter({"max": 250});
});
&lt;/script&gt;
